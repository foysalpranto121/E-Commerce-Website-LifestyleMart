## OPTION 2: Render.com (Free - 3 minutes)

**Step-by-step for live demo:**

1. Go to https://render.com (sign up with GitHub - 30 seconds)
2. Click **"New +"** → **"Web Service"**
3. Connect your GitHub repo
4. Fill these EXACT values:
   - **Name:** `lifestylemart`
   - **Runtime:** `Python 3`
   - **Build Command:** `pip install -r requirements.txt`
   - **Start Command:** `gunicorn app:app`
   - **Plan:** `Free`
5. Click **"Create Web Service"**
6. Wait 2 minutes → Your URL appears!

## OPTION 3: Railway.app (Free - 2 minutes)

1. Go to https://railway.app
2. Login with GitHub
3. Click **"New Project"**
4. Select **"Deploy from GitHub repo"**
5. Choose your repo
6. Railway auto-deploys! ✨

## OPTION 4: PythonAnywhere (Always Free)

1. Go to https://pythonanywhere.com
2. Create account (no credit card)
3. Open **Bash console**
4. Run:
   ```bash
   git clone https://github.com/foysalpranto121/E-Commerce-Website-LifestyleMart.git
   cd E-Commerce-Website-LifestyleMart
   pip install -r requirements.txt
   python database_setup.py
   ```
5. Go to **Web** tab → **Add a new web app**
6. Select **Flask** → Python 3.11
7. Path: `/home/YOURNAME/E-Commerce-Website-LifestyleMart/app.py`
8. Click **Reload** → Done!
